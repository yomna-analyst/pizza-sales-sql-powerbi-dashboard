# 🍕 Pizza Sales SQL & Power BI Dashboard

Sales analytics project built using **SQL** and **Power BI** to analyze pizza sales performance and generate business insights.

---

## 🎯 Objective
Analyze sales data to track revenue, orders, pizza performance, and category trends.

---

## 🧠 Project Workflow
- Cleaned and analyzed sales data using SQL
- Created SQL views for the required business metrics
- Connected the SQL view to Power BI
- Built an interactive sales dashboard

---

## 📊 Dashboard Overview
- Total Sales
- Average Order Value
- Total Pizzas Sold
- Total Orders
- Top 5 Pizzas by Quantity
- Top 5 Pizzas by Revenue
- Sales by Pizza Category

---

## 🛠 Tools Used
- SQL
- SQL Views
- Power BI
- Data Visualization
